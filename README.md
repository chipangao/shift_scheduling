Spring Boot 班表管理系統，提供員工管理與班表排程功能。

## 技術棧

| 技術 | 用途 |
|------|------|
| Spring Boot 3.5.14 | 後端框架 |
| MyBatis 3.0.3 | ORM / SQL 映射 |
| PostgreSQL 15 | 資料庫 |
| Redis 7 | 查詢快取 |
| Docker Compose | 一鍵啟動開發環境 |

## 快速啟動

### 前置需求
- Docker Desktop
- 無需安裝 Java / Maven / PostgreSQL / Redis

### 啟動服務

```bash
docker compose up -d
